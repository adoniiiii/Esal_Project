# Project Idea: Es Al KG

## Project Type

Website / Web Platform

## Problem

Tourists or anyone who want to visit Kyrgyz jailoo areas often cannot find local services online. Yurts, pavilions, horse rentals, local guides and food services are usually not listed on platforms such as Booking.

Because of this, tourists cannot easily plan their trip in advance, and local people lose potential customers.

## Solution

"Es Al" is a centralized eco-tourism booking website that connects tourists with local jailoo service providers.

Tourists can browse and book complete packages that include:

- Yurt, pavilion accommodation
- Horse riding
- Local guide services
- Local meals

## Target Users

- Local tourists
- Foreign tourists
- Families
- Eco-tourism travelers
- Local service providers

## MVP

The MVP will include:

- Home page
- Package catalog
- Package details page
- Booking request form
- AI travel assistant
- Backend API
- PostgreSQL database
- Dockerized setup
