# Бизнес-анализ — Es Al

---

## 1. Problem Statement

**Проблема:**
Иностранные туристы, которые хотят пожить в юрте в Кыргызстане, не могут найти нормальный сайт на английском языке где можно посмотреть варианты и сразу забронировать. Всё происходит через WhatsApp и Instagram — без цен, без гарантий, без удобного поиска.

**Решение:**
«es al» — сайт где иностранный турист может найти юрту или топчан по регионам Кыргызстана, задать вопросы ИИ-чатботу и сразу оформить бронь онлайн.

**Ценностное предложение:**
> "Authentic Kyrgyz experience — booked in minutes, not days."

---

## 2. Целевая аудитория

Единственный тип пользователя — **иностранный турист**.

Владельцы юрт не регистрируются в системе. Их контактные данные добавляет администратор вручную при заполнении карточки объекта. Это осознанное решение — так команда контролирует качество объявлений на старте.

### Персона — Том, 32 года, Германия

| Параметр | Описание |
|---|---|
| Цель | Провести отпуск нестандартно, пожить в юрте на Иссык-Куле |
| Боли | Не знает русского, нет нормального сайта, боится мошенников |
| Поведение | Бронирует всё онлайн заранее, доверяет фото и описаниям |
| Устройство | Ноутбук и смартфон |
| Ожидание | Английский интерфейс, понятные цены, мгновенное подтверждение |

## 3. Анализ рынка и конкуренты

**Ситуация на рынке:**
Туризм в Кыргызстане растёт — особенно экотуризм и этнический туризм. При этом ни одной специализированной платформы для бронирования юрт не существует. Это и есть свободная ниша.

| Конкурент | Чем похож | Почему не закрывает нашу нишу |
|---|---|---|
| Airbnb | Онлайн-бронирование жилья | Почти нет юрт в КР, высокая комиссия |
| Booking.com | Большой каталог | Юрты не представлены, сложная регистрация |
| Instagram / WhatsApp | Там сейчас происходят все брони | Нет бронирования, нет безопасности, нет English UI |
| Visit Kyrgyzstan | Официальный туристический ресурс | Только информация, никакого бронирования |

**Вывод:** Прямого конкурента с фокусом на юрты + иностранный язык + онлайн-бронирование на рынке Кыргызстана нет. «es al» занимает свободную нишу.

## 4. Бизнес-модель и монетизация

**MVP (сейчас):** сервис полностью бесплатный. Контент добавляется администратором вручную.

**После MVP — как будем зарабатывать:**

| Источник дохода | Как работает | Ставка |
|---|---|---|
| Комиссия с брони | Берём % с каждой подтверждённой брони | 5–8% от суммы |
| B2B партнёрства | Интеграция с турагентствами | По договору |

**Цели после запуска:**
- 50+ активных объявлений юрт и топчанов
- 200+ зарегистрированных туристов за первые 3 месяца
- 30+ успешных броней в месяц к концу первого сезона

## 5. Customer Journey Map (CJM)

Путь пользователя: **Том хочет пожить в юрте на Иссык-Куле**

| Этап | Действие пользователя | Экран в «es al» | Боли / Мысли | Решение в «es al» |
|---|---|---|---|---|
| 1. Поиск | Гуглит "yurt rental Issyk-Kul", находит «es al» | Главная страница | «Есть ли нормальные варианты?» | Карта с фильтрами, фото, цены |
| 2. Изучение | Просматривает карточки юрт | Каталог / карточка юрты | «Можно с детьми? Есть электричество?» | Фото, описание удобств, контакты |

| 4. Авторизация | Регистрируется через email | Экран Sign Up | «Ещё один аккаунт, только бы быстро» | Быстрая регистрация, минимум полей |
| 5. Бронирование | Выбирает даты, гостей, жмёт Book Now | Форма бронирования | «А вдруг не подтвердят?» | Мгновенное подтверждение, сводка |
